# Qt_serial_assistant
自制的一个功能较全的串口调试助手开源版本
基于QT Creator IDE开发，使用QMake编译

主要功能如下：

1.基础状态数据包配置与初始化。包括自动检索当前设备串口号，波特率、数据位、停止位、校验位的配置，另附一个指示灯标识当前状态

2.编码类型切换。实现了接收数据和发送数据的编码类型为十六进制或者ASCLL原码任意切换

3.定时发送。实现了自定义定时器并定时发送数据到从机用于连续数据流的发送或测试

4.文件模式的发送与接收。读取当前设备的txt文本并发送，或将当前接收端接收到的数据保存为txt文本格式到当前设备下

5.日志模式。在接收端定时以当前时间戳显示发送或者输出类型的数据信息

6.DTR和RTS模式的配置
